12-Month Development Plan

From Design to Pilot Demonstration

Phase 1

Design and Architecture

Months 0–3

During this phase, WaveShift will finalize the system architecture, select components, and complete detailed design work. This includes defining propulsion, sensors, onboard computing, communications, and autonomy requirements.

Deliverables:

  • Finalized system design
  • Bill of materials
  • Autonomy software framework
Phase 2

Prototype Build

Months 4–6

The physical prototype of Sentinel-ASV will be constructed and assembled. Core systems will be integrated and tested to enable manual and remote operation.

Deliverables:

  • Completed surface vessel
  • Integrated propulsion and sensors
  • Remote operation capability
Phase 3

Autonomy Integration

Months 7–9

Autonomous navigation and safety logic will be implemented and tested. The system will be capable of executing patrol routes and streaming sensor data in real time.

Deliverables:

  • Autonomous waypoint navigation
  • Obstacle avoidance
  • Live sensor data streaming
Phase 4

Pilot Demonstration

Months 10–12

WaveShift will conduct on-water demonstrations to validate system performance and business value. Results will be documented for pilot partners and investors.

Deliverables:

  • On-water pilot demo
  • Performance metrics
  • Pilot readiness documentation
